# 🧪 Lab Supply Finder

**Зертханалық жабдықтар мен реактивтерді іздеу жүйесі**

Лабораториялық жабдықтар, реактивтер және тұтынушы материалдарын іздеу, салыстыру және сұраныс жіберу үшін құрылған веб-қолданба.

## 🚀 Ерекшеліктері

### 1. 🔐 Тіркелу және Кіру
- **Жаңа пайдаланушылар** үшін аккаунт құру функциясы
- **Пайдаланушы валидациясы** (Email форматы, құпиясөз ұзындығы)
- **Құпиясөзді ұмыт қойдың ба?** функциясы
- **LocalStorage**-та деректерді сақтау

### 2. 🔍 Іздеу және Автоматты Ұйсынымдар
- **Нақты іздеу** - жабдықтың атауы, сипаттамасы, өндіруші бойынша
- **Автоматты ұйсынымдар** - іздеу кезінде максимум 8 нәтиже
- **Категория сүзгілеу** - Барлығы, Жабдық, Реактив, Тұтынушы

### 3. ⭐ Таңдаулылар Жүйесі
- **Таңдаулыларға қосу/алу** батырмалары
- **Таңдаулы өнімдер сөлінісі**
- **LocalStorage**-та сақталады

### 4. 🔄 Өнімдерді Салыстыру
- **Максимум 3 өнімді** салыстырып қоюға болады
- **Салыстыру таблицасы** - параллель сипаттамаларды көрсету

### 5. 📬 Сұраныс Жіберу
- **Өнім таңдау ішінде** сұраныс формасы
- **Сана енгізу** және **ескертпе** қосуға болады

### 6. ⚙️ Әкімші Панелі
- **Жаңа өнім қосу**
- **Өнімді өңдеу**
- **Өнімді жою**
- **Өнімдер таблицасы**

### 7. 👤 Пайдаланушы Аккаунты
- **Қолданушының аты және рөлі** сөлінің қоймасында
- **Шығу батырмасы**
- **Әкімші құқықтары**

## 🛠️ Технологиялар

- **React 18+** - интерактивті UI
- **TypeScript** - типті қауіпсіздік
- **Vite** - жылдам құрылым құрушы
- **CSS3** - градиент және анимациялар
- **LocalStorage API** - деректерді браузерге сақтау

## 📝 Ішінгі Қолданбасы

### Іздеу
1. Іздеу өрісіне жабдық аты енгізіңіз
2. Автоматты сұрау қалыңдары пайда болады
3. Категория батырмасын басыңыз (ерекше)

### Таңдаулылар
1. Өнім картасындағы ⭐ батырмасын басыңыз
2. "⭐ Таңдаулылар" бөлімінде көрінетін болады

### Салыстыру
1. 🔄 батырмасын басыңыз (максимум 3)
2. "🔄 Өнімдерді салыстыру" бәлкеме басыңыз

### Сұраныс Жіберу
1. Өнімді таңдаңыз
2. "📬 Сұраныс жіберу" батырмасын басыңыз
3. Сана және ескертпе енгізіңіз
4. "✅ Сұраныс жіберу" батырмасын басыңыз

### Әкімші Функционалы
1. Бәлкемеде ⚙️ батырмасын басыңыз
2. "➕ Жаңа өнім қосу" басыңыз
3. Өнімнің ақпаратын толтырыңыз

## 🔒 Қауіпсіздік Ескертпелері

> **Ескерту:** Бұл демонстрация қолданбасы. Өндіктеу орнатуында:
- API құрылымы үстіне материал трансферты
- Құпиясөз шифрау (bcrypt немесе ұқсас)
- JWT токендар әндігі сеансты басқару
- HTTPS әндігі құрылымы

## 🐳 Контейнеризация (Docker)

1. `docker-compose build`
2. `docker-compose up -d`
3. Фронтенд: `http://localhost:5173`, бэкенд: `http://localhost:3001/api/health`

## 🔁 CI/CD (GitHub Actions)

- Файл: `.github/workflows/ci.yml`
- Әр push/PR үшін:
  - Frontend: `npm ci`, `npm run lint`, `npm run build`
  - Backend: `npm ci`, Sequelize мұрағат синхронизациясы (`sequelize.sync({ alter: true })`)

## 🔧 Тексеру

- Backend: `curl http://localhost:3001/api/health`
- Frontend: `curl http://localhost:3001/api/supplies`
